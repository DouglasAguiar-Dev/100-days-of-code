# ☕ Coffee Machine

A simple command-line **Coffee Machine** built with Python as part of my **100 Days of Code - Python Bootcamp** journey.

This project simulates a real coffee machine where users can order drinks, insert coins, receive change, and consume available resources. It also keeps track of the machine's profit and remaining ingredients.

---

## 🚀 Features

- ☕ Order three different drinks:
  - Espresso
  - Latte
  - Cappuccino
- 💰 Coin payment system
- 💵 Automatic change calculation
- 📊 Resource report (water, milk, coffee, and money)
- ✅ Checks if enough ingredients are available before making a drink
- 🔄 Updates resources after every successful order
- 🔒 Secret command to turn the machine off
