# 100 Days of Code - Python Bootcamp Progress

Welcome to my `100-days-of-code` repository.

This repository documents my journey learning Python through the **100 Days of Code: The Complete Python Pro Bootcamp** by Dr. Angela Yu. It is a public record of my progress, practice, mistakes, improvements, and growth as a learner developer.

My goal is not only to complete exercises, but to build real programming habits: consistency, problem solving, clean code, debugging, and project organization.

## About This Repository

Each folder represents a daily project from the bootcamp. The projects start simple and become more challenging over time.

This repository helps me:

- Track my progress as I learn Python.
- Practice Git and GitHub workflow.
- Build small projects consistently.
- Improve my problem-solving skills.
- Show my learning history as I grow toward becoming a software developer.

Some early projects are intentionally simple. They are part of the learning process and show the foundation I am building step by step.

## Learning Goals

- Understand Python fundamentals clearly.
- Write readable and maintainable code.
- Practice clean naming, structure, and indentation.
- Learn how to debug problems independently.
- Build projects that demonstrate real programming concepts.
- Develop consistency through daily practice.
- Create a portfolio that shows my progress over time.


## Skills I Am Building

- Python syntax and fundamentals
- Variables, strings, numbers, and data types
- Conditional logic
- Loops and iteration
- Lists and dictionaries
- Functions and parameters
- Basic game logic
- Debugging and testing ideas manually
- Git and GitHub version control
- Project organization

## My Learning Approach

For each project, I focus on understanding the logic before improving the code. I try to learn from errors, refactor when possible, and make each project a little cleaner than the previous one.

This repository is part of my long-term goal of becoming a professional software developer. I am using it to build discipline, confidence, and a visible history of my learning.
