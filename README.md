# Treasure Island

## About This Project

This is my third Python project, created while following the **100 Days of Code: Python Bootcamp** course by **Angela Yu**.

Treasure Island is a simple text-based adventure game where the player must make a series of choices to find the hidden treasure. Depending on the choices made, the player can win the game or reach one of several game over scenarios.

## What I Learned

During this project, I practiced:

* User input with `input()`
* Conditional statements (`if`, `elif`, and `else`)
* Nested decision making
* Program flow control
* Variables
* String comparison
* Basic game logic
* Reading and understanding code structure

## Features

* Interactive text adventure game
* Multiple paths and outcomes
* Winning and losing scenarios
* ASCII art welcome screen
* Simple decision tree gameplay

## Technologies

* Python 3
