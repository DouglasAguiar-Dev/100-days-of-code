Tip Calculator
About This Project

This is my second Python project, created while following the 100 Days of Code: Python Bootcamp course by Angela Yu.

The goal of this project is to calculate how much each person should pay when splitting a bill, including a tip percentage.

I used this project to practice programming fundamentals and improve my understanding of how programs are structured.

What I Learned

During this project, I practiced:

Variables
User input
Data types (int and float)
Mathematical calculations
Conditional statements (if / else)
Loops (while)
Error handling (try / except)
Functions (def)
Return values
Input validation
Features
Calculates the total bill with tip included
Splits the bill between multiple people
Validates user input
Prevents invalid values such as negative numbers
Handles text input errors using exception handling
Repeats questions until valid input is entered
