# Rock Paper Scissors

A simple Rock Paper Scissors game built with Python.

This project was developed as part of the **100 Days of Code: Python Bootcamp** by Angela Yu. The purpose of the project was to practice Python fundamentals while building a complete command-line game.

---

## Features

- Player name input
- Rock, Paper, Scissors game logic
- Random computer choice
- ASCII art for each option
- Win, lose, and tie conditions
- Invalid input detection

---

## Technologies Used

- Python 3
- Random Module

---

## Concepts Practiced

- Variables
- User Input
- Lists
- Conditional Statements (`if`, `elif`, `else`)
- Comparison Operators
- Logical Operators (`and`, `or`)
- Random Number Generation
- Program Flow Control

---

## How to Play

1. Run the program.
2. Enter your name.
3. Choose:
   - `0` for Rock
   - `1` for Paper
   - `2` for Scissors
4. The computer will randomly make a choice.
5. The game will display the winner.
