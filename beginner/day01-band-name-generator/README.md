# 🎸 Band Name Generator

A simple Python command-line application that creates a fun band name by combining your favorite city with your pet's name.

This project was built as part of my **100 Days of Code – The Complete Python Pro Bootcamp** by Angela Yu. While the original exercise focuses on basic user input and variables, I expanded it by adding input validation, improved user interaction, and the ability to generate new band names without restarting the application.

---

## ✨ Features

* Prompts the user for their favorite city.
* Prompts the user for their pet's name.
* Validates user input to prevent empty entries.
* Automatically formats names using title case.
* Generates a band name using Python f-strings.
* Allows the user to create new band names in the same session.
* Handles invalid menu selections with clear feedback.
* Lets the user exit the program safely.
